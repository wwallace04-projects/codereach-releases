Terminal access to every machine
from your iPhone

CodeReach runs a lightweight daemon on each host, giving you full tmux-powered terminal sessions from a native iOS app — over LAN or from anywhere via gateway tunnels.

How It Works

1

Install the daemon

One package installs everything: binary, tunnel client, service file. Auto-starts on boot.

2

Pair with the app

Scan the QR code or copy the token into the CodeReach iOS app.

3

Connect

Tap a host card to open a full terminal session. Works over Wi-Fi or cellular.

iPhone (CodeReach app) | |--- LAN (Wi-Fi) -------> daemon :9090 on any reachable machine | |--- SSH tunnel -------> Gateway -------> daemon (via frp reverse tunnel) (works anywhere) (FreeBSD) (Linux, macOS, WSL2)

What's in the Package

codereach-daemon Go binary, ~11 MB
frpc Reverse tunnel client
Service file systemd / launchd
Auto-configure Token, gateways, startup
